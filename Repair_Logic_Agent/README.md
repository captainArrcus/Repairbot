We are building a agentic rag tool, with the purpose of it to give good and clear instructions on how to repair everything. 

The code is structured in a 
- data_crawling_pipe, to find all kinds of manuals and data_sheets of different products, we need a competent data crawling pipeline

- agents, for the different agents, working on it

- agent_tools, for all the tools, the agents can use in the future

- retrieval_pipeline, for the embedding and retrieval steps

- utils, for all the other utils of the base programm
